
# [Robots Vs. Zombies](https://robots-vs-zombies.now.sh/)

Top-Down, Multi-player, Zombie Survival Shooter built with Phaser & Socket.io.

How long can you survive?

## Real-Time Infrastructure

*  Phaser.io
*  Socket.io
*  EventEmitter3
*  Node.js
*  Express
*  Webpack
*  Docker
*  Zeit Now
